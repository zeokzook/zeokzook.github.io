CONTRIBUTIONS

Resource Area Spawning

I developed the resource and building spawning system for the game. We wanted resource areas and buildings to spawn around the player. So, I made a system where in the beginning of the game, resource areas and buildings will spawn around the player in a circle around the spaceship. On a new day, old resource areas will despawn and respawn around the spaceship. Resource Areas will not spawn close around the spaceship but will spawn outside of a certain radius that can be modified.

After we got some meshes made by our artist and the terrain of the map done, some additional changes need to be done. Since the terrain in our game is bumpy, the resource points will rotate accordingly based on the normal of the terrain. This is done by doing a line trace and getting the normals from that.

The resource points that spawn are randomly selected from a data table. This data table holds data for the mesh, the resource amount and the type. I made it into a data table so anyone can add different variants of resource areas without the need of a programmer. All that needs to be done is adding a new row and setting the new mesh and resource amount.
